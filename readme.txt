Nuotrauku albumas
Albumas(pavadinimas, aprašymas, data...) -> Puslapis(išdėstymas) -> Nuotrauka(data...)
Vartotojas 	-> Administratorius: vartotojų valdymas
		-> Narys: albumų valdymas
		-> Svečias: pagrindinis puslapis

Plėtimuisi: 	nuotrauku redagavimas, komentavimas, filtrų pritaikymas
 		temos parinkimas
		bendradarbiavimas redaguojant
		prenumeravimas
